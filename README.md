每項最高4分，最低0分\

效能：\
模組化：\
程式碼的編排：\
有意義的命名：\
註解：\
審查意見：

```diff
+ 範例

- 注意！評分項目的冒號後面，只要打上一個數字就好，不要加任何其他字元！
```
